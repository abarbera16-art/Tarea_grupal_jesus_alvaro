# Proyecto 04: Sistema de Notificaciones

## Descripción del Proyecto
Este repositorio contiene la resolución del Proyecto 04 asignado para la Tarea Grupal de Entornos de Desarrollo. El proyecto original consistía en un sistema de envío de notificaciones que presentaba varios problemas de diseño (Code Smells) como cadenas mágicas y código condicional complejo. 

Mediante el uso de herramientas de refactorización del IDE, hemos aplicado patrones como la extracción de constantes, extracción de métodos y el patrón Strategy (Polimorfismo) para obtener un Clean Code.
