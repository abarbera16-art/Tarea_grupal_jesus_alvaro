# 🛠️ Proyecto de Refactorización y Clean Code (Project04)

## 📖 Descripción del Proyecto
Este repositorio contiene la práctica grupal de refactorización de código en Java. El objetivo principal ha sido mejorar la clase `NotificationService` eliminando los "Code Smells". Hemos aplicado principios de Clean Code, diseño orientado a objetos y control de versiones colaborativo utilizando Git Flow.
